> This application allows a user to upload photos and description. The user can pick one as a favorite picture. It stores the photos to AWS S3 service. Home screen displays all the stored photos.

## Install

```sh
yarn install 
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn test 
```
